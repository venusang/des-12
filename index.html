Adding an Image

  1. In the formatting menu, click on the [ image] icon.
  2. Locate the image and then click Upload Image
  3. After the system has confirmed that the image has been upload, click Insert
  4. Click on the image to adjust sizing and alignment

Adding and setting up an Answer Type

An item may include one or more answer types. For example, it may include a Short Answer question and an Open Ended question to ask the student to explain his or her answer.

  1. In the formatting menu, click on the Answer Type drop down menu
  2. Select the answer type you would like to add to the item
  3. In the editor menu, click on the [pencil image] to open the configuration panel
  4. Add the information (e.g., choices, feedback, scoring information)
  5. Click Done to return to the editor

The Preview window will be automatically visible so that you can view your changes in real time.

Adding Math using LaTex or MathML

  1. In the formatting menu, click on the [math] icon.
  2. Add text in the form of MathML or LaTex to the window.
    1. If needed, refer to LaTex or MathML authoring sites that are referenced in the configuration panel
    2. If using LaTex, indicate whether the text should be rendered as Block (stand alone on one line) or Inline with text.
  3. Click Done to return to the editor.

Adding Video

Authors may add videos hosted by YouTube and Vimeo. Authors may add a video by providing the video URL and description below.

  1. In the formatting menu, click on the [video image] icon.
  2. Add a URL for the video.
  3. Adjust the aspect ratio to standard or wide screen.
  4. Select the size of the video.

Adding Video

In Categorize, students may drag & drop answer tiles to the appropriate category area or areas.

  1. In the formatting menu, click on the Answer Type drop down menu and select Categorize.
  2. In the editor menu, click on the to open the configuration panel
  3. Determine the categories for the interaction.
    1. Add Categories using the "Add a Category" button.
    2. Remove Categories using the .
  4. Edit the choices.
    1. Adding a Choice: Use the "Add a Choice Button" to add a choice.
    2. Removing a Choice: Use the [trash can] next to the choice
  5. Determine the correct answer. Do this by selecting the category into which the choice should be placed using the drop down menu next to the choice.
  6. Author the feedback by selecting the preferred settings:
    1. Correct Answer: If the interaction if fully correct; the default message is "Correct!"
    2. Partially Correct Answer: Author feedback message if the interaction is partially correct: the default message is "Almost!"
    3. Incorrect Answer: Author feedback message if the interaction is partially correct: the default message is "Good try but that is not the correct answer."
    4. Summary Feedback: Authors may provide interaction level feedback for the answer type. The default setting is set to "no feedback."
  7. Score the interaction. Allow partial credit based on the number of correct answers submitted, this feature is optional.
  8. Define other settings
    1. Shuffle: Selecting "Shuffle Choices" will change the order of choices as they are presented to a student each time that the item is presented launched.
    2. Labeling
      1. Choice Group Label: Authors may label the container for the choices; the default is no label.
      2. Category Labels: Authors may label the area where the student will place the choices in the correct order; the default is 'no label.'

Creating an Item

When you create a new item it'll be named "Untitled" by default. To rename the item:

  1. Go to the File menu.
  2. Select Rename.
  3. Type a name for your item.
  4. Click OK.

Adding a title will update the title that will be used in the Item Profile.

To create an item:

  1. Enter an introduction and/or a prompt and format it using standard formatting features such as italics, bold, headings.
  2. Select the way that you want the student to respond by inserting an Answer Type into the item by using the drop down menu in the formatting toolbar.
  3. Edit the answer type by opening the configuration panel
